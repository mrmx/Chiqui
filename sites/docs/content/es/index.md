Un generador de sitios estáticos ligero, basado en contenido, construido sobre SvelteKit.

## Características

- **Markdown-first** — Escribe contenido en archivos `.md`, obtén un sitio web completo
- **Multi-idioma** — i18n integrado con IDs canónicos para enlaces entre idiomas
- **Validación primero** — Problemas de contenido detectados en build, no en runtime
- **DaisyUI + Tailwind** — Estilos por defecto, totalmente personalizables
- **Config tipada** — Tu configuración se valida en tiempo de desarrollo
