A lightweight, content-driven static site generator built on SvelteKit.

## Features

- **Markdown-first** — Write content in `.md` files, get a full website
- **Multi-language** — Built-in i18n with canonical IDs for cross-language linking
- **Validation-first** — Content issues caught at build time, not runtime
- **DaisyUI + Tailwind** — Beautiful defaults, fully customizable
- **Type-safe config** — Your site config is validated at dev time
